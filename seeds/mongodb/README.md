# Seeds de MongoDB

Datos de prueba para colecciones de MongoDB.

## Estructura

- `material_summaries.js` - Resúmenes de materiales
- `assessments.js` - Resultados de evaluaciones

## Carga

Los seeds se cargan automáticamente con:
```bash
./scripts/setup.sh --seed
```

## Formato

Archivos JavaScript que se ejecutan con mongosh.
